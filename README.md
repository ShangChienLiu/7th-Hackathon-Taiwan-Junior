## 📍 Overview

The 7th-Hackathon-Taiwan-Junior project is a PHP-based web application that aims to provide a platform for offering services such as digital device rental and remote teaching, aimed at helping rural areas or economically challenged groups. Through our services, we aim to bridge the educational gap between urban and rural areas, ensuring that everyone has equal access to education.
